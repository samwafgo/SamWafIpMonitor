# SamWafIpMonitor

## 介绍
SamWafIpMonitor是一个监控本地向外访问的80/443端口的安全辅助小工具。
 

![封面](img/cover.png)
## 功能
可实时查看：
- 发起IP
- 发起程序名称
- 发起程序所在位置
- 目标IP
- 目标端口

## 技术细节
- 程序基于.net core 6,
- 依赖 老九 封装库 [WindivertDotnet](https://github.com/xljiulang/WindivertDotnet)
